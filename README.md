# 🏠 House Price Prediction using Ensemble Learning

A Machine Learning web application that predicts house prices using Ensemble Learning (Gradient Boosting Regressor). The application is built with Flask and features a modern responsive user interface.

## Features
- House Price Prediction
- Ensemble Learning
- Flask Backend
- Responsive UI
- User Input Prediction
- Deployed Online

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS
- JavaScript
